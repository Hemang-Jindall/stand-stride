# Stand & Stride

A mobile-first internship management system built with React, TypeScript and Vite.

## Features

- Student Dashboard
- Admin Dashboard
- Attendance
- Grievance Management
- Certificates
- Mentor Assignment
- Venue Assignment
- Master Data Management

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
